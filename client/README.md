
## Job Tracker Frontend
Welcome to the Job Tracker frontend! This React application provides a user interface for tracking job applications, interviews, and related activities during your job search process.

### Getting Started
To get started with developing the frontend, follow the steps below:

### Prerequisites
Make sure you have the following software installed on your system:

* Node.js (including npm)

### Installation
1. Clone this repository to your local machine:

```
https://github.com/derussoj/cs467_job_tracker.git
```
2. Navigate to the client directory:
```
cd cs467_job_tracker/client
```
3. Install dependencies:
```
npm install
```
### Usage

1. Start the development server:

```
npm start
```

2. Open your web browser and visit http://localhost:3001 to access the Job Tracker frontend.

### Development
The frontend is built with React.
React components are located in the src directory.
Stylesheets and assets can be found in the src directory.
